# C-socket-program
  tcp/udp send and receive  
## compile:  
  gcc lab1c.c -o lab1c  
## execute:  
 *  ./lab1c tcp send &lt;ip&gt; <port> <input filename>  
 *  ./lab1c tcp recv <ip> <port> <output filename>  
  * ./lab1c udp send <ip> <port> <input filename>  
  * ./lab1c udp recv <ip> <port> <output filename>
